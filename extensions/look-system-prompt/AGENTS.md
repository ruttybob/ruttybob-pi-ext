# AGENTS.md — look-system-prompt

- Команда `/system-prompt` — полноэкранный оверлей с просмотром текущего системного промпта и схем инструментов.
- Табы: System Prompt | Mode Preview | Active Tools | Tool Schemas (←→ для переключения).
- Mode Preview показывает, что активный mode/preset внедрит перед следующим LLM-вызовом.
- Tool tabs: Enter раскрывает/сворачивает отдельные инструменты.
- Использует pi-tui компоненты: Markdown, Container, Text, Key.
