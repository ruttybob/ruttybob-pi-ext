# AGENTS.md — tools

- `/tools` — interactive checkbox UI for enabling/disabling tools.
- `/tools <group>` — toggle whole group on/off.
- `/tools-group` — manage groups (create, delete, toggle).
- Ignore patterns: `toolignore.json` (global or project) hides tools from selector UI only — does not affect registration.
- Groups persist in `.pi/toolgroups.json`.
